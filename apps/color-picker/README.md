## カラーピッカー＆パレットの仕様

### 基本機能
- RGB / HSV スライダーで色を操作
- リアルタイムでプレビューカラーが変わる
- 気に入った色をパレットに保存
- パレットから色を選択して再編集

### UI構成
- カラープレビュー（現在の色を大きく表示）
- RGBスライダー：`R` `G` `B` 各0〜255
- HEXコード表示（例：`#FF5733`）
- `パレットに追加`ボタン
- 保存済みカラーパレット一覧

### 動作仕様
- スライダーを動かすとリアルタイムでプレビューが更新される
- RGB ↔ HSV は内部で相互変換して同期する
- `パレットに追加`ボタンで現在の色をパレットに保存
- パレットの色をクリックするとスライダーがその色に切り替わる
- パレットは最大8色まで保存できる
- パレットが満杯の時は一番古い色が削除される

### あえて含めないもの
- HEXコードの直接入力
- パレットの個別削除
- データの永続化

## カラーピッカー＆パレットのUIデザイン

### レイアウト全体像

```
┌─────────────────────────────────┐
│      🎨 カラーピッカー           │  ← タイトル
├─────────────────────────────────┤
│  ┌───────────────────────────┐  │
│  │                           │  │
│  │       カラープレビュー      │  │  ← 現在の色で塗りつぶし
│  │                           │  │
│  └───────────────────────────┘  │
│  HEX: #FF5733                   │  ← HEXコード表示
├─────────────────────────────────┤
│  R  [━━━━━━━●━━━━━] 255        │
│  G  [━━●━━━━━━━━━━] 87         │  ← RGBスライダー
│  B  [●━━━━━━━━━━━━] 51         │
├─────────────────────────────────┤
│       [パレットに追加]           │  ← ボタン中央揃え
├─────────────────────────────────┤
│  パレット                        │
│  [■][■][■][■][■][■][■][■]  │  ← 最大8色
└─────────────────────────────────┘
```

### 各要素の詳細

| 要素 | 詳細 |
|------|------|
| ウィンドウサイズ | 幅400 × 高さ500 px |
| カラープレビュー | 高さ120px、`egui::Painter`で塗りつぶし |
| HEXコード | プレビューの下に中央揃えで表示 |
| RGBスライダー | ラベル ＋ スライダー ＋ 現在値（数値） |
| 追加ボタン | 中央揃え |
| パレット | 正方形（40×40px）を横並びに最大8個 |
| パレットの色 | クリックでスライダーに反映、枠線あり |

### カラーイメージ

- 背景：eguiデフォルトのダークテーマ
- プレビュー枠：白い枠線で囲む
- パレット枠線：未選択はグレー、選択中は白
- スライダー：eguiデフォルト
