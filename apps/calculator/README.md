# 電卓アプリ

## 計算機の仕様

### 基本機能
- 四則演算（`+` `-` `×` `÷`）
- 小数点入力
- 計算結果の表示
- `C`ボタンで全リセット

### UI構成
- ディスプレイ部分（入力中の数値・結果を表示）
- 数字ボタン：`0`〜`9`
- 演算子ボタン：`+` `-` `×` `÷`
- 実行ボタン：`=`
- クリアボタン：`C`

### 動作仕様
- 数字を押すとディスプレイに追加される
- 演算子を押すと左辺値を記憶して、次の数値入力待ちになる
- `=`を押すと計算を実行して結果を表示する
- ゼロ除算時はディスプレイに `Error` と表示する
- 結果表示後に数字を押すと新しい入力として扱う

### あえて含めないもの
- 連続計算（`1 + 2 + 3`のような chain 計算）
- キーボード入力対応
- 履歴機能
